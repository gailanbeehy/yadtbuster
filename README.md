<h2> Installation cmd </h2>

* `apt update && apt upgrade`
* `pkg install git`
* `git clone https://github.com/princekrvert/tbuster.git `
* `cd tbuster`
* `chmod +x *`
* `./install.sh`
* `./tbuster.py --help`
